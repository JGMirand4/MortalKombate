# Mortal Kombate

## Descrição

Este projeto consiste em um jogo de luta entre lutadores. Os lutadores são representados por uma estrutura chamada Lutador, que possui informações como nome, ataque, defesa, vida e número de vitórias. Os lutadores são gerados aleatoriamente e distribuídos em dois times para a batalha.

## Funcionalidades

- Geração aleatória dos lutadores.
- Distribuição dos lutadores em dois times.
- Simulação de batalha entre os times.
- Determinação do vencedor com base na vida dos lutadores.

## Como executar

1. Clone o repositório para a sua máquina local.
2. Compile o programa usando um compilador C (por exemplo, gcc).
3. Execute o programa resultante.
4. Siga as instruções exibidas no console para interagir com o jogo.

## Exemplo de Uso

Após executar o programa, será exibida uma tela com os times de lutadores que irão batalhar. O jogador poderá pressionar a tecla 'enter' para iniciar a luta. A cada rodada, os lutadores atacarão alternadamente até que um time seja derrotado. O programa indicará o vencedor ou a derrota do seu time, caso ocorra.
